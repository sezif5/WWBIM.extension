# ДГП Уровни

## Описание

Скрипт копирует названия уровней из параметра **ДГП_Имя уровня** в параметр **Имя** для всех уровней в модели.

## Принцип работы

1. Скрипт собирает все уровни (OST_Levels) в модели
2. Для каждого уровня:
   - Читает значение из параметра `ДГП_Имя уровня`
   - Записывает это значение в параметр `Имя`
   - Если значения совпадают или `ДГП_Имя уровня` пусто — уровень пропускается
3. Формируется отчет о выполнении

## Параметры

- **Исходный параметр**: `ДГП_Имя уровня` — текстовый параметр уровней
- **Целевой параметр**: `Имя` — встроенный параметр Revit для уровня (текстовый)

## Отчет

После выполнения скрипт выводит:

### Общая статистика
- Всего уровней
- Обновлено
- Пропущено

### Почему уровни пропускались
- Пустое значение в ДГП_Имя уровня
- Не найден целевой параметр
- Параметр только для чтения
- Значение уже совпадает
- Неверный тип параметра
- Ошибка при записи

### Детали по уровням
Таблица с результатами для каждого уровня (до 100 строк):
- ID — кликабельный идентификатор элемента
- Уровень — название уровня
- Результат — ✓ Обновлено / ✗ Пропущено
- Причина — описание причины пропуска

## Использование

1. Запустить инструмент из Revit
2. Дождаться завершения и изучить отчет

## Примечания

- Скрипт работает только с категорией **Уровни** (Levels)
- Параметр `Имя` — это встроенный параметр Revit для уровней
- Все ID в таблице кликабельные для быстрого перехода к уровню в Revit

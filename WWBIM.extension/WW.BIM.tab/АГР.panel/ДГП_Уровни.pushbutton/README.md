# ДГП Уровни

## Описание

Скрипт меняет местами значения параметров **ДГП_Имя уровня** и **Имя** для всех уровней в модели.

## Принцип работы

1. Скрипт собирает все уровни (OST_Levels) в модели
2. Для каждого уровня:
   - Читает значение из параметра `ДГП_Имя уровня`
   - Читает значение из параметра `Имя`
   - Записывает значение из `ДГП_Имя уровня` в параметр `Имя`
   - Записывает значение из `Имя` в параметр `ДГП_Имя уровня`
   - Пропускает уровень, если оба параметра пустые
3. Формируется отчет о выполнении с указанием «было → стало» для каждого параметра

## Параметры

- **Параметр 1**: `ДГП_Имя уровня` — текстовый параметр уровней
- **Параметр 2**: `Имя` — встроенный параметр Revit для уровня (текстовый)

## Отчет

После выполнения скрипт выводит:

### Общая статистика
- Всего уровней
- Обменено
- Пропущено

### Почему уровни пропускались
- Оба параметра пустые
- Не найден параметр
- Параметр только для чтения
- Значения уже одинаковые
- Ошибка при записи

### Детали по уровням
Таблица с результатами для каждого уровня (до 100 строк):
- ID — кликабельный идентификатор элемента
- Уровень — название уровня
- Результат — ✓ Обменено / ✗ Пропущено
- Причина — описание причины пропуска
- ДГП_Имя уровня → `было → стало`
- Имя → `было → стало`

## Использование

1. Запустить инструмент из Revit
2. Дождаться завершения и изучить отчет

## Примечания

- Скрипт работает только с категорией **Уровни** (Levels)
- Параметр `Имя` — это встроенный параметр Revit для уровней
- Если какой-то параметр недоступен (не найден или только для чтения), его значение не меняется
- Все ID в таблице кликабельные для быстрого перехода к уровню в Revit
